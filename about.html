About the Project

Classic Arcade Games, Reimagined for the Web

VANILLA is a web-based arcade hub that brings timeless classic games into the modern era. Built with pure HTML5, CSS3, JavaScript, and a Python Flask backend, this project showcases how retro gaming experiences can be recreated with contemporary web technologies while maintaining the essence of what made these games legendary.

üõ†Ô∏è Technology Stack

This project demonstrates modern web development practices while maintaining clean, readable code without heavy frameworks.

HTML5 Canvas CSS3 Vanilla JavaScript Python 3.8+ Flask JSON API
  • Frontend: Pure HTML5 Canvas for game rendering, CSS3 for UI styling, and vanilla JavaScript for game logic
  • Backend: Python Flask server providing REST API endpoints for game data, leaderboards, and procedural generation
  • Persistence: LocalStorage for client-side high scores, JSON file store for global leaderboards
  • Responsive Design: Adaptive canvas sizing and mobile-friendly controls
  • Performance: 60 FPS target with optimized rendering and efficient game loops
